# UrjaSaathi 💡🇮🇳  
**AI-Powered Smart Electricity Awareness System for Middle-Class Indian Households**

![UrjaSaathi Banner](https://user-images.githubusercontent.com/banner-image-placeholder)

## 🚀 Project Overview

UrjaSaathi is an AI + IoT-based mobile and web platform designed to empower middle-class Indian families to track, control, and optimize their electricity usage. It delivers real-time consumption insights, appliance-level monitoring, and personalized energy-saving suggestions — all in regional languages.

The platform connects users to government electricity subsidy schemes (like PM Surya Ghar), helps prevent bill shocks, and encourages smarter energy habits through AI coaching and gamified community engagement.

## 🧠 Key Features

- ⚡ **Real-Time Electricity Tracking**  
  Plug-and-play smart meter or ESP32 integration to visualize live kWh and ₹ usage.

- 🧩 **Appliance-Level Monitoring**  
  Track individual devices with smart plugs (AC, fridge, fan) to identify energy hogs.

- 🤖 **AI Energy Assistant**  
  Get daily energy-saving suggestions, abnormal usage alerts, and usage forecasts.

- 📢 **Voice Alerts in Regional Languages**  
  AI assistant speaks Hindi, Tamil, Bengali, etc. to support non-English users.

- 🏠 **Subsidy Eligibility & Application**  
  Auto-check for state/central electricity subsidy schemes with one-click apply.

- 🥇 **Community Leaderboard**  
  Compete with neighbors or localities to be the most energy-efficient household.

- ☀️ **Solar & EMI Recommendation Tool**  
  Personalized solar panel sizing and payback calculator with financing options.

## 🛠️ How It Works

1. **Install Smart Plug or ESP32 Module**  
   Plug into wall + configure Wi-Fi. Device sends power data via MQTT.

2. **User Login & Sync**  
   Mobile/web app connects via secure API to sync data per user household.

3. **AI Assistant & Alerts**  
   Backend runs ML models to detect abnormal spikes or waste. Users get alerts + voice advice.

4. **Subsidy Finder**  
   Based on usage + location, the app surfaces relevant subsidy schemes and links to apply.

5. **Leaderboard**  
   Community stats drive healthy competition, energy points, and monthly badges.
